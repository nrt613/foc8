# Foc8 — Free ADHD Pre-Screening Test

Foc8 is a free, science-based ADHD pre-screening tool that combines:

- A real **CPT (Continuous Performance Test)** — the same type used in clinical settings, not just another quiz
- **Validated questionnaires** — Vanderbilt for children, ASRS-5 for adults
- A personalized report showing where you land vs. age norms

## Try it

👉 **[foc8.netlify.app](https://foc8.netlify.app)**

12 minutes, at home. One-time payment for the full report — no subscription, no hidden fees.

## Important

Foc8 is a pre-screening tool, **not a diagnosis**. It's designed to help you have a more informed conversation with a doctor — not replace one.

## Languages

Available in English, Hebrew, Spanish, Arabic, French, and German.

## Tech

Single-file HTML/JS application. Built solo over a year of development and validation.
